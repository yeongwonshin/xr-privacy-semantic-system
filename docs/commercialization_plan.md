# Commercialization Plan

## 1. Positioning

**"AI Glasses용 Privacy Firewall + Semantic Compression SDK"**로 포지셔닝한다. 단순 XR 앱이 아니라 XR 제조사와 기업 고객이 붙일 수 있는 보안 인프라로 판다.

## 2. 제품 패키지

### Starter SDK
- Edge semantic extraction
- PII redaction
- basic policy YAML
- developer audit log

### Enterprise SDK
- tenant policy management
- advanced sensitive object detector
- VLM Proxy
- admin dashboard
- SOC2/ISO27001 evidence export

### On-prem Suite
- private VLM adapter
- SIEM 연동
- MDM/SSO 연동
- 고객사 내부망 배포

## 3. 가격 가정

- Developer: 무료/월 0원, packet quota 제한
- Pro: 디바이스당 월 과금
- Enterprise: 연간 계약 + 구축비
- On-prem: 구축비 + 유지보수

## 4. 초기 타깃 시장

1. 산업 안전 스마트글래스
2. 의료 교육/수술 보조 XR
3. 공공기관·보안구역 현장 점검
4. 교육 XR 플랫폼

## 5. 투자자에게 보여줄 데모

- 같은 프레임에서 raw image upload 없이 VLM 답변 생성
- 전화번호/카드/얼굴 자동 제거
- semantic packet size vs raw frame size 비교
- 감사 로그 hash chain 검증
- 정책을 바꾸면 cloud 전송이 차단되는 장면
