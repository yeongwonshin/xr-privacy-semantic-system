# 5-Minute Demo Script

## 0:00 문제 제기
"스마트글래스가 계속 카메라를 켠 상태로 클라우드 VLM에 화면을 보내면, 편리함과 동시에 개인정보 문제가 폭발합니다."

## 0:40 핵심 아이디어
"이 시스템은 원본 영상을 보내지 않습니다. 사용자의 의도에 필요한 텍스트, 객체, 레이아웃만 추출해서 semantic packet으로 보냅니다."

## 1:30 데모 1: 문서 요약
전화번호가 포함된 안전 매뉴얼을 입력한다. Edge에서 전화번호가 마스킹되고 서버에는 `[PHONE_KR_REDACTED]`만 전달된다.

## 2:30 데모 2: 객체 안내
작업자 얼굴과 카드가 보이는 장면에서 민감 객체는 suppress되고 control panel, warning label만 전송된다.

## 3:30 데모 3: 감사 로그
서버 API 호출 뒤 audit tail을 확인한다. packet 생성, 정책 평가, VLM 응답이 해시 체인 로그로 남는다.

## 4:20 사업화
"XR 앱 하나가 아니라, XR 기업들이 반드시 붙여야 하는 privacy firewall SDK로 팔 수 있습니다."

## 5:00 결론
"AI glasses 시대에는 카메라를 켜는 기능보다, 무엇을 서버에 보내지 않을 것인가가 경쟁력이 됩니다."
